OUTPOST 404

Remote survival colony ops in your terminal. Balance scarce resources, survive random events, and complete the Signal Beacon before the outpost collapses.

Mission

Win
  • Survive 45 days — reach day 46 with vitals above zero
  • Signal Beacon — complete 5 parts before collapse
Lose
  • Power, food, morale, or population hits zero
  • Collapse is checked before a survival win on the final day

Install

Homebrew

brew install ericdahl-dev/tap/outpost-404

Go

go install github.com/ericdahl-dev/outpost-404/cmd/outpost@latest

From source

git clone https://github.com/ericdahl-dev/outpost-404.git
cd outpost-404
go run ./cmd/outpost

Run

$ outpost
# fixed seed for reproducible events
$ outpost -seed 42
# optional JSONL session log
$ outpost -log ./logs/my-run.jsonl

Controls

Key Action

Headless tools

Balance scripts and session replay for tuning and regression checks:

outpost -simulate scripts/conservative.json -seed 99
outpost -simulate scripts/survival_45.json -seeds 1,7,42
outpost -replay ./logs/my-run.jsonl

See the repo balance guide for reference seeds and baseline expectations.

Stack

Built with the Charm stack: Bubble Tea, Lip Gloss, and Bubbles. Game rules live in internal/game; the TUI in internal/ui. Content is JSON-driven (data/buildings.json, data/events.json).
